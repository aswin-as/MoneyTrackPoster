MONEY TRACK
Voice-Enabled Expense Tracker using Python
Abstract
Money Track is a Python-based voice-enabled expense tracker that lets users record daily spending quickly through speech or typing. All expenses are saved in a JSON file, and the system provides summaries and charts to help users easily understand their spending habits without complex database setups.

About The Project
The Money Track App is an offline Python application designed for financial awareness.

  • Offline Capability: No internet needed.
  • Voice Command: "Spent 500 on Food".
  • Secure: Local JSON storage ensures privacy.
  • Visuals: Matplotlib graphs for analysis.
Software Requirements
  • OS: Windows / Linux / macOS
  • Language: Python 3.x
  • Development: VS Code / PyCharm
  • Libraries: JSON, Matplotlib, SpeechRecognition, Vosk, Scipy
