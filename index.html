MONEY TRACK
Voice-Enabled Expense Tracker using Python
Abstract
Money Track is an offline Python-based expense tracking system designed to help users monitor and manage their daily financial activities efficiently. The application supports both manual typing and voice-based input using the offline Vosk speech recognition engine, allowing users to record expenses quickly and conveniently. All data is stored permanently in a local JSON file, which acts as the system’s lightweight dataset and removes the need for an internet connection or external database. The project also includes features such as daily and monthly summaries, category-wise spending analysis, and visual charts to help users clearly understand their financial behavior. Money Track aims to offer a simple, private, and user-friendly platform for individuals who want an effective way to track their expenses offline.

About The Project
Money Track is an offline expense management application created using Python to provide users with a smooth and practical way to track their day-to-day spending. The system allows users to record expenses either by typing or through voice commands using Vosk, enabling hands-free and faster data entry. All recorded information is stored securely in a JSON file, which serves as the project’s dataset and ensures that users can access their data without internet connectivity. The application also generates summaries, category-based reports, and visual charts to give users better insights into their spending habits. By combining simplicity, offline functionality, and voice-enabled features, Money Track provides a convenient and reliable tool for students and general users to manage their expenses effectively.
